# Shipyard — Agent Discovery

## What this is

Shipyard is a folder-based AI coaching system. Drop the files into any AI assistant that supports file context (Claude, ChatGPT, Gemini, etc.) and it becomes "the Shipwright" — a coach that guides beginners through shipping their first real AI project.

## Entry point

Start with `identity.md`. It routes to `rules.md` and the appropriate phase folder in `reference/`.

## File map

| File | Purpose |
|------|---------|
| identity.md | Coach persona, phase routing table |
| rules.md | Step protocol, coaching constraints, stage map |
| examples.md | Example coaching interactions per phase |
| reference/phase-0-settle/ | Phase 0: Defuse overwhelm |
| reference/phase-1-find-your-edge/ | Phase 1: Surface builder identity |
| reference/phase-2-pick-one/ | Phase 2: Lock one project |
| reference/phase-3-define-done/ | Phase 3: Define smallest working version |
| reference/phase-4-build/ | Phase 4: Coach through blockers |
| reference/phase-5-ship/ | Phase 5: Push through stalls |
| reference/phase-6-show/ | Phase 6: Write the story |

## How to use

1. Upload all `.md` files and the `reference/` folder to your AI assistant
2. Start a new conversation — the coach activates automatically
3. Follow its lead — it will guide you through the phases
