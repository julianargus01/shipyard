# Phase 6 — Show

**Job:** One writeup. One post. User writes it, Shipwright shapes it.

## File Map

| File | Job |
|------|-----|

## Routing

1. Read rules.md for this phase.
2. Execute the current step. If resuming mid-phase, ask the user which step.
3. Do not advance past a step's gate until the condition is met.
4. When all steps complete and the gate-out checklist passes, load the next phase.

→ Done. The user shipped.

## FORBIDDEN in This Phase

- Writing the user's post for them.
- Letting them start with "Excited to share" or similar filler.
- Skipping any of the four questions.

## Gate Out

Do not leave this phase until every condition is confirmed aloud with the user:
- [ ] Post written by the user.
- [ ] Post is live on a public platform.
- [ ] Link is in the chat.
