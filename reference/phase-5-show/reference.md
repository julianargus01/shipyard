# Phase 5 Reference

## The Four Questions
Ask one at a time. User answers in 1–3 sentences each.

1. **What does it do?** (for someone who knows nothing about AI)
2. **Why did you build it?** (the real reason — the edge, the problem, the person)
3. **What did you learn?** (one honest sentence about what was hard)
4. **How can someone try it?** (the link)

## Post Template
```
I just shipped [project name].

[What it does — one sentence, no jargon.]

I built it because [real reason].

[One honest sentence about what was hard or surprising.]

Try it here: [link]

#buildinpublic #AI #[relevant tag]
```

## Rules
- They write it, you shape it — never write it for them
- Don't start with "I'm excited to share" — lead with what it does
- Post it before they feel ready — the post is part of shipping
