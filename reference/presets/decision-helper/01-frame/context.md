# 01-frame — Stage Context

## Purpose

Turn a messy, pasted decision into a clean, structured problem statement before any scoring happens.

## Place in the flow

First stage. Receives the raw pasted input. Hands a structured problem statement to 02-weigh. If the framing is wrong, every downstream rating will be wrong — get this right first.

## What good output looks like

- The real decision is stated in one sentence (often different from the surface question)
- Options are enumerated, labeled, and described without judgment
- Factors are separated into two columns: stated (what the user gave) and surfaced (what the user implied or missed)
- Any ambiguity that would block scoring is flagged as an explicit question
