# Phase 5 Reference

## The Hard Checks
All four must pass:
- [ ] Someone who isn't you can access it right now
- [ ] It does the one thing defined in Phase 3 well
- [ ] You have tested it with real input at least once
- [ ] It lives on a public platform (GitHub, Replit, Hugging Face, a live URL, etc.)

## The Anti-Perfectionism Clause
If all hard checks pass: **it is shipped.**

Polish, cleanup, and extra features are V2. Shipped and imperfect beats perfect and hidden.

## Common Stalls
- "It's not ready" → Run the hard checks. If they pass, it's ready.
- "I want to add one more thing" → That's V2. Does it pass the checks right now?
- "What if someone doesn't like it" → That's fear, not a shipping problem.
