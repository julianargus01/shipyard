# Phase 1 — Pick ONE

**Job:** Present the 4 presets, get the user to pick one, and frame it as a working shape they make their own.

---

### Step 1 — Teach + Present Options

- **Teach:** "We're going to build a workflow based on a folder structure. The folders contain files with step by step instructions for an agent to follow, so you don't have to prompt it at every step. I'm going to give you some workflows to pick from, which you will personalize. Your job in this session is to put your voice, your angle, and your judgment into it so it runs the way you would run it. Choose from one of these:"

  1. **File Organizer** — turn a messy file list into a clean structure. *(Paste a file list.)*
  2. **Brain-Dump Planner** — turn everything on your mind into today's plan. *(Paste a brain dump.)*
  3. **Notes** — turn raw notes into a clean recap. *(Paste messy notes.)*
  4. **Decision Helper** — turn a stuck decision into a clear recommendation. *(Paste a decision + options.)*

- **Wait.** Do not proceed until the user names one.
- **Review:** The user names exactly one preset.
- **Correct (too many / can't decide):** "Which pain do you face most often? That's your pick." → re-present the options.
- **Correct (none resonate):** "Tell me what you'd want the tool to do. I'll map it to the closest preset."
- **Approve (lock the pick + frame the build):** "[name] it is. It already has a working shape — you do not build it from a blank page. Your job is to make it yours: we go through it one piece at a time and put in your voice, what a good result looks like for you, and the rules it uses to decide. Then we run it on your own input. Let us make it yours."

**Gate:** Do not leave this phase until ONE preset is named by the user AND the working-shape framing has been delivered.
**If gate fails:** Present one preset at a time and ask whether it fits.

Folder name mapping (coach-internal — never spoken; silently note the chosen one):
- File Organizer → `file-organizer`
- Brain-Dump Planner → `brain-dump-planner`
- Notes → `notes`
- Decision Helper → `decision-helper`
