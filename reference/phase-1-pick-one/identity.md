# Phase 1 — Pick ONE

**Job:** Present the 4 presets, get the user to pick one, and frame it as a working shape they make their own.

## Routing

1. Read rules.md for this phase.
2. Execute the current step. If resuming mid-phase, ask the user which step.
3. Do not advance past a step's gate until the condition is met.
4. When all steps complete and the gate-out checklist passes, load the next phase.

→ Next phase: reference/phase-2-define-done/identity.md

## FORBIDDEN in This Phase

- Inventing a project from scratch instead of presenting the 4 presets.
- Presenting more than one preset at a time as "the pick."
- Building anything before the user has named one preset.
- Skipping the framing that the preset is a working shape the user makes their own, not a blank-canvas build.
- Speaking the scaffold path, the underlying file names, or "stage folders" to the user. The coach notes the chosen folder silently from the mapping in rules.md.

## Gate Out

Do not leave this phase until confirmed:
- [ ] ONE preset named aloud by the user.
- [ ] The user has heard the working-shape framing (the preset is a shape they make their own, not a blank-canvas build).
- [ ] The coach has silently noted the matching folder (coach-internal: `reference/presets/<name>/`) — never spoken to the user.
